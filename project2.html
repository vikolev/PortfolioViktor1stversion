Group Project Pizzeria

Title: "Pizzeria Picasso Group Project"

Date: 20rd October

Ended 9th November

Project Description: During the orientational phase, my university gave us an assignment in which we were split into groups and had the job to make a working website for a Pizzeria where we needed to implement a smart oven using an Arduino board. We were given 5 weeks for these assignments. My job for the project was to work on the HTML, the CSS and with the Arduino with the help of one of my groupmates!
